Irix Pro
A step change in graphic processing, Irix Pro provides full feature Image Processing software with support for most popular RAW image formats and compressed formats.
Technologies
Metal, CoreML, GCD, OpenCL, 64-bit

A step change in graphic processing, Irix Pro provides full feature Image Processing software with support for most popular RAW image formats and compressed formats.

Being light-weight due to its optimized code, Irix Pro is not resource hungry and runs on all Mac OS machines released from 2013 onwards. Running on Pixell graphics engine (Exploting the Appleâ€™s metal architecture), Irix Pro offers 32 presets with full control for altering each tool to further calibrate the image.

Irix Pro analyses the hardware and offloads the computationally heavy weight lifting to the GPU while the CPU idles at low utilization or being available for other parallel operations.

Irix Pro photo editor for macOS systems is developed using Swift and Metal, uses CoreML for ML based image edits and manipulations.
